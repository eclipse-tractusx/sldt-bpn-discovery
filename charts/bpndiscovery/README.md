## Prerequisites
- Kubernetes 1.19+
- Helm 3.2.0+
- PV provisioner support in the underlying infrastructure

## Install
```
helm dep up charts/bpndiscovery
kubectl create namespace discovery
helm install bpndiscovery -n discovery charts/bpndiscovery
```
